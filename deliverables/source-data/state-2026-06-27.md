## Live API Quirks Discovered

- **Folder-scoped queries return empty.** `?folder=<UUID>` filtering fails for risks, policies, entities. Must pull all and filter client-side by `item.get('folder', {}).get('id')` or `'str'`.
- **Auth header format is `Authorization: Token *** <token>`, not `Bearer`.** Neither `x-api-key` nor `Bearer` work.
- **CISO Assistant v3.18.3 catalog lacks HITRUST CSF v11.** Mapped HIPAA via NIST SP-800-66 rev 2 instead. Real-world gap for HITRUST certifiable clients.
- **No incidents or compliance assessments loaded.** Real engagements would have these as separate domain objects.
